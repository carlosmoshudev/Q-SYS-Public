UCI QR Code Tool

Use this plugin to generate QR codes for up to 20 user control interfaces (UCIs) in a design. When a user scans a QR code from a UCI using a mobile device, the UCI then opens on the device â€“ either the Q-SYS iOS App or the browser web interface in Q-SYS Core Manager. This eliminates the need to physically interact with a touch screen peripheral.

Compatibility

The plugin is compatible with Q-SYS Designer version 8.3.1 and above.

Configuration Overview

  1. Drag the plugin into the schematic.
  2. Configure the plugin properties. See Properties.
  3. Press F5 to save your design to the Core and run it. (Or, press F6 to emulate your design.)
  4. In the Setup tab, select the network interface and method for launching the UCI on the mobile device. See Controls.
  5. In the QR Codes tab, you will see QR codes for each UCI in your design. (If the tab is blank, see the Troubleshooting section.)
  6. Press F7 to stop running or emulating the design.
  7. Copy a UCI's QR code control into its respective UCI. You can do this by either clicking and dragging the QR code box into the UCI's tab or selecting, copying, and pasting.
  8. Run or emulate your design again.

The UCI now displays the generated QR code. Users can now scan this code to open the UCI within a mobile device.

Note: The mobile device must be on the same network as the Q-SYS Core.

Properties

Debug Print

When the 'Show Debug' property is enabled, select what information is displayed in the Debug Output window:

  • None: (Default) Nothing is printed.
  • UCIs: Prints the discovered UCIs in the design.
  • Function Calls: Prints a statement denoting which function has been called.
  • HTTP Code: Prints the code returned from HTTP downloads.
  • All: Prints all statements from the plugin.

Show Debug

Select 'Yes' to show the Debug Output window. Select 'No' (default) to hide the Debug Output window. For details, see the Debug Output topic in the Q-SYS Help.

Controls

Setup

Network Interface

Select what network interface is used for communication with the plugin. Available selections depend on whether the design is running on the Core (F5) or emulating (F6):

  • If running on the Core, the Core's LAN interfaces are displayed.
  • If emulating, the PC's interfaces are displayed.

Note: If the method of deployment changes, this control prompts for a new selection. If it remains the same when re-deploying, the last selected interface is retained and QR codes are re-generated on startup.

UCI Device

Select how the UCI launches on the mobile device:

  • iOS App: Launch the UCI in the Q-SYS Control iOS app.
  • Web UCI: (Default) Launch the UCI in the web browser interface in Q-SYS Core Manager.

QR Codes

QR Code (Image)

Up to 20 QR codes can display in this tab, depending on the number of UCIs found in the design. Drag or copy/paste these codes into your UCIs.

Note: QR codes are regenerated if the Setup controls change and when restarting the design.

Troubleshooting

UCI not appearing after scanning the QR code

The mobile device must be on the same network as the Q-SYS Core.

QR Codes tab is blank

The QR Codes tab does not display QR code images if no UCI exists in your design.

Note: You must have a User Control Interface (UCI) deployment feature license to deploy a UCI to the Q-SYS Core.
