Shure Microflex Wireless

Use this plugin to connect to and monitor a Shure Microflex Wireless (MXW) device. You can also identify a channel (if supported by the device) and toggle muting of a particular channel.

Note: See the Microflex Wireless product page on the Shure website for product information and documentation.

Compatibility

  • The plugin is compatible with all models within the MXW series.

  • One plugin instance per device is supported.

Prerequisites

Use the MXW control software to configure the device before using the plugin:

  • In the Preferences tab of the MXW GUI, set the "Mute Preference" to "External Mute" to enable plugin control of the mute state.

  • To control the LEDs of the microphone, make certain that "External LED Control" is selected in the MXW GUI.

Properties

Is Managed

Select 'Yes' to add the plugin to your Inventory list, which makes the plugin's Status available for monitoring in Core Manager and Enterprise Manager. When enabled, you can then specify a Name and Location for the plugin, as with any other Inventory item.

For more information, refer to the Plugins topic in the Q-SYS Help.

Show Debug

Select 'Yes' to show the Debug Output window. For details, see the Debug Output topic in the Q-SYS Help.

Debug Print

Set the data printed to the Show Debug window.

  • None: (default) None of the below will be posted to the log debug output.

  • Tx/Rx: Transmission and Received Data messages will be shown.

  • Tx: Transmission Data messages will be shown.

  • Rx: Received Data messages will be shown.

  • Function Calls: Each function within the plugin will post a message when it is called.

  • All: All of the above messages.

Connection Setup

Unit/IP

Specify a device IP address here. One plugin instance per device is supported.

Status

Once a Mode and Unit/IP are selected, the plugin will automatically attempt a connection. If you see "OK" status, you are successfully connected to the device. If you see a "Fault" or "Missing" error, check the connection parameters.

Details

Once the plugin connects, the detected unit's name and control IP are shown here.

ID

Once the plugin connects, toggle this button to cause an LED to blink on the connected device (if supported by the model).

Channel Controls

Note: Some controls will show as disabled if not applicable to the device/model.

Channel Name

Displays the read-only channel name as set in the MXW control software.

Identify

Toggle this button to cause an LED to blink on the respective channel.

Battery Charge

Displays the level of charge on the respective channel’s battery. This control is disabled if the battery charge is 0 or is not applicable to the device.

Audio Meter

Displays the input level of the connected microphone on that channel, from -60 to 0 dB.

RF

Shows the signal strength of the receiver to the base unit.

Transmitter Status

Displays the read-only status of the respective transmitter. Status colors and messages can be adjusted in the MXW control software.

Mute

Toggles muting of the respective transmitter/channel.
