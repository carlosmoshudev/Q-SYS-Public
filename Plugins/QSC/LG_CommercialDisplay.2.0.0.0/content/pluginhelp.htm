LG Commercial Display

Use this plugin to connect to, control, and view the status of an LG Commercial Series Display.

Note: See the LG Website for product information and documentation.

Compatibility

The plugin was developed and tested with Q-SYS Designer Software version 9.4.1 and LG 43UL3J-E and 49UH5F-HJ displays running firmware version 03.06.72. The plugin supports LG commercial displays with the serial API support. This includes models that allow access to the API on port 9761 or via serial RS232 connections.

Note: LG Commercial Signage displays have varying functionality between model families. The owner's manual for your display should have a "Command Reference List" that lists the API commands available for the plugin to connect with. When configuring the plugin, you may need to disable or enable properties that relate to the available commands for your display.

Prerequisites

Before using the plugin, you must set up the LG Display.

  • Set up the display with either a serial connection from the Q-SYS Core processor's RS232 port to the display or a network connection that is routed so that the Core and Display are on the same network segment.
  • Set and note the IP address and Display ID of the display.

Connecting to the Device

After setting up the LG Display, drag the plugin into the schematic to begin using it.

  1. From the Properties pane, select the Connection Type as Serial or Network. If Serial, drag the serial component for the Core into the design and connect the serial pins between the Core and the plugin.

  2. From the Properties pane, set the number of Inputs for the plugin to display. Since the LG Display may have more inputs than physical connectors, set this value to at least double the number of physical input connectors. For a description of all configurable properties, see Properties.

  3. Set the available properties for the functions that the plugin will use for your display. You can activate or disable Wake on Lan methods, Audio Controls, Display Information requests, Screen Status (backlight) requests, Input Signal Status requests, Temperature requests, Video Wall information and temperature requests, and whether or not to show a discovery mechanism.

  4. Press F5 to save your design to the Core and run it.

  5. Double-click the plugin to open the control panel.

  6. In the plugin's Connection Setup tab, enter the IP address of the LG Display (if in Network mode).

  7. Alternatively, if the Discovery property was enabled, go to the Discovery tab and press "Send Discovery" to search the local network for displays. The combobox of Displays Found will populate with displays that responded. Selecting a display will set its IP address on the Connection Setup tab.

  8. Enter the Display ID.

  9. The plugin should now connect to the display and populate the Device Information. If Information is not loaded, verify the Display is on, the IP Address is correct, and the Set ID matches the Display.

  10. Switch to the Controls tab. For the Inputs, click the Scan Inputs button to have the plugin try each input and populate the Input slots. Note that this will cause the TV to rapidly change between inputs. Alternatively, click the Input Name drop-down to assign the Input slot manually.

  11. Several input types accessible on the Display may be related. Select the Link field to link that input to a different Input slot. When linked, the plugin will report the target of the link as the active input when the linked Input Slot is reported by the Display.

    Tip: Linking can be used to consolidate DTV and PC type inputs that are for the same physical connector into one button/indicator.

The plugin will automatically attempt a connection once the IP Address is entered, or immediately on boot for Serial connections. If you see "OK" status, you are successfully connected to the device. If you see a "Fault" error, check to make sure you entered the correct parameters.

Network Connection Requirements

For Network connected plugins:

  • The plugin must connect to the TV once before the power-on function will work.

  • Power-on command in Network mode requires the Display to allow Wake On LAN and requires the plugin to retrieve the MAC Address from the device. This setting for Wake On LAN mode is displayed on the Connection Setup page for convenience.

  • LG Displays will not respond to Network requests when in a powered off state. When the LG Display is powered off the plugin will report the display as missing.

  • When connecting an Ethernet display to a Q-SYS Reflect-managed design, use the Screen Power Off control to turn off the screen instead of powering off the entire display. This maintains the connection and avoids the plugin reporting a MISSING status. (Reflect may send warnings and emails when the display is powered off.)

Properties

Is Managed

Select 'Yes' to add the plugin to your Inventory list, which makes the plugin's Status available for monitoring in Core Manager and Enterprise Manager. When enabled, you can then specify a Name and Location for the plugin, as with any other Inventory item.

For more information, refer to the Plugins topic in the Q-SYS Help.

Connection Type

Select 'Network' or 'Serial' for the connection method the plugin will use to access the Display.

Inputs

Set the number of Input slots for the plugin to display on the Controls page. Each Input slot is connected to an input mode reported by the display. These can be automatically scanned or manually configured on the Controls page. Many input types on the Display may be available for each physical connector, as well as virtual input types. Set the Inputs number high enough to make sure there are inputs for all your uses.

Poll Rate

Set the number of seconds between data poll requests sent from the plugin to the display.

Power On Delay

Set the number of seconds after sending a power on command that the plugin should ignore communication errors. Many displays will disconnect while booting, use this to suppress errors in communication and status states during the boot process. Integer between 0 and 60.

Model Family

Set the Model Family of your display. This will apply a known, good configuration for displays from this series.

  • Default: No configuration applied. All properties are available for manual configuration.

  • UR640: allows Audio Controls, Get Screen Status. Removes Get Display Info, Get Signal Status, Get Temperature, Video Wall.

  • UH5F/UL3J: allows Audio Controls, Get Screen Status, Get Display Info, Get Signal Status, Get Temperature. Removes Video Wall.

  • Video Wall (VM5E): allows Audio Controls, Get Screen Status. Get Display Info, Get Signal Status, Video Wall. Removes Get Temperature.

  • TR3BF: allows Audio Controls, Get Screen Status, Get Display Info. Removes Get Temperature, Get Signal Status, Video Wall.

Note: TR3BF models use a separate API and will not be compatible with most LG displays.

Wake On Lan

Configure the Wake on Lan mode for the plugin. When Power On is sent, a Wake On LAN packet is sent in addition to API commands (only in Network mode).

  • Off: Wake On Lan packets will not be sent.

  • Broadcast: A Wake On Lan broadcast packet will be sent. This is usually the most supported mode. A 255.255.255.255 broadcast is used.

  • Local Network: A local network broadcast address is calculated and used for the Wake On Lan packet. In the case of a 24-bit 192.168.1.?, this will broadcast to 192.168.1.255 .

  • Static Address: The Wake On Lan packet will be broadcast to the device's IP address.

Audio Controls

Select 'Yes' to show Audio Controls and request mute and volume data from the display.

Get Display Info

Select 'Yes' to show Serial Number and Software version on the Connection Setup page. Some displays do not provide this information over the API.

Get Screen Status

Select 'Yes' to show Screen Backlight controls for On, Off, Status and power %.

Get Signal Status

Select 'Yes' to show Signal Status indicator for the current input.

Get Temperature

Select 'Yes' to show the display's temperature on the Connection Setup page.

Tip: Some displays do not report this information and removing this will make the connection more stable.

Video Wall

Select 'Yes' to show the Ports and Cabinets Per Port property. When more than 0 of each are entered, a page of Cabinet Temperatures will be added to the plugin for monitoring the individual temperature of each display in the video wall, as well as the overall maximum temperature.

Ports

When Video Wall is set to 'Yes', enter the number of Ports in the Video Wall's setup.

Cabinets Per Port

When Video Wall is set to 'Yes', enter the number of Cabinets connected to each Ports in the Video Wall's setup.

Discovery

Select 'Yes' to add a Discovery page to the plugin. The Discovery page will allow users to search the local network for displays and select the display for connection from a combobox list. To be discovered, LG displays must be powered on and connected to the same network as the Q-SYS Core processor.

Show Debug

Select 'Yes' to show the Debug Output window. For details, see the Debug Output topic in the Q-SYS Help.

Debug Print

Set the data printed to the Show Debug window.

  • None: (default) None of the below will be posted to the log debug output.

  • Tx/Rx: Transmission and Received Data messages will be shown.

  • Tx: Transmission Data messages will be shown.

  • Rx: Received Data messages will be shown.

  • Function Calls: Each function within the plugin will post a message when it is called.

  • All: All of the above messages.

Controls

Connection Setup

IP Address

The IP address of the LG Display on the network. Not shown in Serial operation mode.

Reset Serial

Trigger button to disconnect and restart the serial connection between the Core and the Display. Not shown in Network operation mode.

Set ID

The LG Display's ID. Must match the configured value on the Display. Used to index specific Displays when multiple displays are chained together via Serial or Ethernet daisy chain.

Connection Status

Displays the current connection status:

  • OK: Plugin is connected to the unit
  • COMPROMISED: Plugin is connected to the unit; Device is reporting errors.
  • FAULT: Check the credentials to connect to the device.
  • MISSING: No device is found at the specified IP address.

Serial Number

Read only text displaying the serial number of the display.

Software Version

Read only text displaying the current software version installed on the display.

MAC Address

Text displaying the MAC Address of the display. The MAC Address will be retrieved on connection to the display, or can be input manually to use Wake on LAN functionality before connecting.

Temperature

Read only number reporting the current temperature reported by the display in degrees Fahrenheit.

Controls

Power Toggle

A toggle button that sends the power on command when clicked in low state, or the power off command when clicked in high state. The State is set when the display is polled for the current power status.

Power On

A trigger button that sends the power on command to the display.

Power Off

A trigger button that sends the power off command to the display.

Power Status

Current reported power state of the display. High for power on.

Screen Toggle

A toggle button that sends the screen on command when clicked in low state, or the screen off command when clicked in high state. The State is set when the display is polled for the current screen status. Screen and backlight power off may not be available for all Display models.

Screen On

A trigger button that sends the screen on command to the display. Screen and backlight power off may not be available for all Display models.

Screen Off

A trigger button that sends the screen off command to the display. Screen and backlight power off may not be available for all Display models.

Screen Status

Current reported screen state of the display. High for screen on, low is for screen off/standby. Screen and backlight power off may not be available for all Display models.

Brightness

Fader to set the display's backlight power. Allowed values are 0-100.

Volume

Fader to set the volume level of the display. Allowed values are 0-100.

Mute

Toggle button to set and indicate the current Mute state for the display. A high value indicates the mute state is active.

Current Input

Combo box. The value is set to the current input reported by the display. Selecting a different input will send the set input command to the display.

Current Input Signal

Indicator LED is set high when the display reports the current selected input has an active video signal. Input signal is not available on all display models.

Input Name

Combo box indicates the name of the input type the input slot is bound to. The slot's Select button and Status indicator will activate and report for the input defined in this field.

Input Select

Trigger button sets the display to the input type listed in Input Name.

Input Status

LED Indicator sets to high when the input listed in Input Name is reported as currently active by the display.

Link

Combo box binds the input listed in Input Name to a different input slot. When an index is set, the plugin will evaluate the Input Name in this slot and set the linked input slot as active.

Mute Video

A toggle button that Mutes the current video feed of the display. The screen will remain on and menu systems will still function. This feature is not available on all display models.

Scan Inputs

A trigger button to have the plugin automatically set up the input slots. The plugin will clear the list of input slots, then rapidly attempt all input types on the display. When an input type is accepted the next available input slot will be configured to use that type. Note that the display will switch to each input during this process. (The display should not be used for live presentation during this process.)

Remote

The remote page displays a series of trigger buttons operating as a remote control for the display. These controls provide no feedback. Not all commands assigned to a remote button are available on all display models. The TR3BF model provides a 12 button remote, while other model families provide a 71 button remote.

Cabinet Temperatures

When the Video Wall property is enabled, this page creates a grid of Ports and Cabinets per Port sized as per the properties.

Celsius/Fahrenheit

Select the units of the displayed temperatures.

Temperature (grid)

Numerical text control, read only. The reported temperature from each cabinet in the display wall will be displayed.

Max

Numerical text control, read only. The highest reported temperature from any cabinet is displayed.

Discovery

When the Discovery property is set to 'Yes', this page is displayed.

Send Discovery

Trigger button. When pressed, the plugin will send an LG UDP discovery packet broadcast across the network.

Displays Found

Combobox text control. Any display that responds to the LG UDP discovery request (sent above) will be listed as an option. Selecting a display from the combobox list will set its IP Address for the plugin's connection and start communication for the plugin with that display. Set ID must still be configured correctly on the Connection Setup page for displays that support Set ID.
