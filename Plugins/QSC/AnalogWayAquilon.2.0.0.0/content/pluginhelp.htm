Analog Way Aquilon

Use this plugin to connect to and control Analog Way Aquilon devices.

The plugin is compatible with Q-SYS Designer version 9.7 and above, and was tested with LivePremier™ firmware version 4.0.254 Beta

If you are using LivePremier™ firmware version lower than 4.0, you must use version 1 of the Analog Way Aquilon Plugin.

NOTE:  See the Analog Way LivePremier Presentation Systems Page on the Analog Way website for product information and documentation.

Prerequisites

Before you can configure the plugin, the Aquilon(s) must be connected to a network and have a valid IP address in a subnet which the Q-SYS core can communicate with.

The IP address can be obtained or configured from the Aquilon device via the WebRCS interface.

It is recommended to configure a static IP address for the Aquilon, as DHCP addresses can change.

Configuration Overview

  1. Drag the plugin into your schematic.
  2. Single-click on the plugin and set the desired number of presets in the Properties settings.
  3. Press F5 to save your design to the Core and run it. (Or, press F6 to emulate your design.)
  4. In your schematic, double-click the plugin to open the component's control panel.
  5. On the Setup tab, in the Setup section, enter the IP Address, MAC Address in the appropriate fields. Select the Interface from the drop-down menu.
