Extron SMP 351

Use this plugin to connect to and control any Extron SMP 351 device, as well as display device status and information.

The plugin is compatible with Q-SYS Designer version 7.2 and above, and was tested with Extron firmware version A1YPNCK.

Note: See the Extron website for product information and documentation.

Plugin Features

With this plugin, you can:

  • Show device information, including storage information.
  • Switch inputs and mute channels.
  • Set transport controls and recording information.
  • Set the test pattern.
  • Set video presets.

Prerequisites

Before using the plugin to control the Extron SMP 351, the device must be powered up and reachable on the network. Refer to the SMP 300 Series Setup Guide on the Extron website for guidance on initially configuring the SMP 351.

Configuration Overview

Connecting to the Device

In the Connection Setup section:

  1. Type the IP Address of the device.
  2. Specify the device Port. This is the Telnet (Extron SIS) port, as configured in the device's web-based user interface within the Configuration > System Settings tab. Typically, this is port 23.
  3. Specify your Password credential for the admin account, and then press Enter.

The plugin will automatically attempt a connection. If you see a "Fault" error, check to make sure you entered the correct parameters.

Obtaining Device Information

After a successful connection to the device, you can view the device's name, model, firmware version, and serial number. This information updates automatically if anything changes on the device.

Controlling the Device

Click the Device Control tab to set device inputs, mute channels, and set recording parameters.

  • Inputs that are currently selected are highlighted in white. To change inputs, click an input button.
  • Video and audio channels that are currently muted are highlighted in white. Click a channel button to mute or unmute that channel.
  • If the device is currently recording, the Record button is highlighted in red. You can also pause, stop, and create chapter markers (if applicable).

Setting Video Configuration

Click the Video Configuration tab to view and set the streaming test pattern, as well as set the video presets. Click Refresh Names to sync the list of options with the device.

Note: The Video Presets section does not show the currently-selected presets on the device. Use this section to set new Layout and Encoder A and B parameters.

Properties

These properties exist in Q-SYS 8.2.0 and later.

Is Managed

For QSC-authored plugins only, select 'Yes' to add the plugin to your Inventory list, which makes the plugin's Status available for monitoring in Core Manager and Enterprise Manager. When enabled, you can then specify a Name and Location for the plugin, as with any other Inventory item.

For more information, refer to the Plugins topic in the Q-SYS Help.

Show Debug

Select 'Yes' to show the Debug Output window. For details, see the Debug Output topic in the Q-SYS Help.

Controls

Connection Setup

IP Address

Type the IP address of the Extron SMP 351 on your network.

Port

Type the Telnet (Extron SIS) port, as configured in the device's web-based user interface within the Configuration > System Settings tab. Typically, this is port 23.

Password

Type the password credential for the device's admin account.

Device Information

Device Name, Device Model, Firmware, Serial Number

This information updates automatically if anything changes on the device.

Storage: Internal, Front USB, Rear USB, RCP USB

Shows the currently available, used, and total storage (MB) for the internal and connected USB storage on the device.

Connection Status

Shows the current status of the device. Normally, this shows "Initializing" for a few moments after running your design, followed by "OK - Telnet Connected". If you see a "Fault" status, check your connection parameters.

Device Control

Note: When you toggle a control, it can take a few seconds for the device to reflect the change.

Inputs: Channel A, B

The current input is highlighted in white. Click a button to change the input.

A/V Mute: Video Channel A, B

Video channels that are currently muted are highlighted in white. Click a channel button to mute or unmute that channel.

A/V Mute: Audio Left, Right Channel

Audio channels that are currently muted are highlighted in white. Click a channel button to mute or unmute that channel.

Record, Pause, Stop

Use these buttons to control device recording. If the device is currently recording, the Record button is highlighted in red.

Mark

Click Mark to create a chapter marker (if applicable). These markers or chapters are viewable in the Extron Media Player (EMP).

Elapsed Time, Time Remaining, Destination

Displays information about the current recording, including the storage destination. Elapsed Time and Time Remaining are refreshed in real time. The Time Remaining value may fluctuate depending on the amount of video information being recorded.

Video Configuration

Note: The Video Presets section does not show the currently-selected presets on the device. Use this section to set new Layout and Encoder A and B parameters.

Test Pattern

View and set the current test pattern.

Layout

Select a preset for the video layout.

Encoder A, B

Select a preset for each encoder channel.

Streaming A, B

Select a preset for each streaming channel.

Refresh Names

Click to sync the list of presets with the device.
