On Screen Keyboard

The On Screen Keyboard plugin is a tool that enables User Control Interface (UCI) users to type keystrokes directly within a UCI on a Q-SYS TSC touch screen controller.

Compatibility

The plugin was developed and tested with Q-SYS Designer Software version 9.0 and also tested to work with version 8.4.

Configuration Overview

To begin using the plugin, drag it into the schematic.

  1. By default, a String Output control pin is exposed. Wire this pin to a scripting or control component – for example, Custom Controls.

  2. Configure the scripting or control component to display a text box.

  3. Double-click the plugin to open its control panel.

  4. Drag the keyboard buttons into a UCI.

  5. If desired, drag the scripting or control component text box into the UCI, which allows the user to see a keystroke echo.

  6. Press F5 to save your design to the Core and run it. Users can now use the keyboard in the UCI to input keystrokes directly from the TSC touch panel.

Properties

Show Debug

Select 'Yes' to show the Debug Output window. For details, see the Debug Output topic in the Q-SYS Help.

Controls

Alphanumeric Keys

Letter keys

These are lowercase by default and uppercase as an alternate.

Number keys

These are numbers by default and the displayed character as an alternate.

Special Characters

Characters

The bottom character is displayed by default. The upper character is displayed as an alternate.

Space

An ASCII space character.

Tab

An ASCII horizontal tab character.

Control Keys

Backspace

Removes the last singular character on with each press. Clears the string output if Shift is engaged.

Enter

Inserts a carriage return, line feed to the string output.

Shift

A single press enables shift for next press. A quick double-press latches the shift function until pressed again. The small LED in the lower-left corner of the left Shift button denotes the shift state.

Caps Lock

A single press toggles Caps Lock on or off. The LED in the lower-left corner of the button denotes the current Caps Lock state.
