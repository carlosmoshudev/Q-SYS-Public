PixelFLEX NovaStar H-Series

Use this plugin to connect to, control, and view the status of a PixelFLEX NovaStar H-Series video splicing processor.

Note: See the NovaStar website for product information and documentation.

Compatibility

The plugin was developed and tested with Q-SYS Designer Software version 9.5.0 and the NovaStar H5 model. This plugin is incompatible with Q-SYS Designer Software versions below 9.4.0.

Prerequisites

Before using the plugin, you must set up the NovaStar H-Series processor. The plugin supports UDP or RS-232 connection. Manuals for the various NovaStar H-Series models can be found here.

  • If using UDP, ensure the processor is reachable on the network.

  • If using RS-232, you can connect to any Q-SYS device that has an RS-232 connector. Serial settings for the NovaStar processor can be set in the web UI or from the front panel. Ensure that the Serial Port IN and the Serial Port OUT settings on the NovaStar processor match.

Tip: For best performance, we recommend using a higher baud rate such as 115200 or 57600.

Configuration Overview

Drag the plugin into the schematic to begin using it.

  1. From the Properties pane:

    • Select either UDP or RS-232 for Control Mode. If RS-232, drag your Q-SYS device's Serial Port component into your schematic and wire it to the plugin's serial pin.

    • Select which Model you are connecting to.

    • Select the Number of Presets as well as the Number of Layers to control from the plugin.

  2. Press F5 to save your design to the Core and run it. (Or, press F6 to emulate your design.)

  3. In your schematic, double-click the plugin to open its control panel.

    • For UDP, on the Setup tab, select the local NIC to use for connection and input the IP address of the NovaStar processor to connect to.

    • For RS-232, on the Setup tab, set the Baud Rate, Data Bits, Stop Bits, and Parity to match what is set on the NovaStar processor.

The plugin will automatically attempt a connection once the IP Address is entered for UDP and once any of the serial settings are changed for RS-232. If you see "OK" status, you are successfully connected.

Properties

Is Managed

Select 'Yes' to add the plugin to your Inventory list, which makes the plugin's Status available for monitoring in Core Manager and Enterprise Manager. When enabled, you can then specify a Name and Location for the plugin, as with any other Inventory item.

For more information, refer to the Plugins topic in the Q-SYS Help.

Control Mode

Select 'UDP' or 'RS-232' for the connection method the plugin will use to access the processor.

  • When set to UDP, the 'Connect to Device' section on the Setup tab of the plugin will have NIC and IP Address controls.

  • When set to RS-232, the 'Connect to Device' section on the Setup tab of the plugin will have Baud Rate, Data Bits, Stop Bits, and Parity controls. This mode will also expose a serial input pin that must be wired to the serial output pin of the Q-SYS device's Serial Port component.

Model

Set this to the model of the NovaStar H-Series processor that you will be connecting to.

Number of Presets

Set this to the number of presets you would like to control from the plugin, 1-16. This will affect the number of available controls on the Presets tab of the plugin.

Number of Layers

Set this to the number of layers you would like to control from the plugin, 1-16. This will affect the number of available controls on the Layer Routing tab of the plugin.

Show Debug

Select 'Yes' to show the Debug Output window. For details, see the Debug Output topic in the Q-SYS Help.

Debug Print

Set the data printed to the Show Debug window.

  • None: (default) None of the below will be posted to the log debug output.

  • Tx/Rx: Transmission and Received Data messages will be shown.

  • Tx: Transmission Data messages will be shown.

  • Rx: Received Data messages will be shown.

  • Function Calls: Each function within the plugin will post a message when it is called.

  • All: All of the above messages.

Controls

Setup

NIC

For UDP Control Mode only. Choose a network interface for the connection:

  • If you are running your design on the Core, this defaults to the active LAN connection on the Core.
  • If you are emulating your design, choose a local network interface on your computer.

IP Address

For UDP Control Mode only. Enter the IP Address of the NovaStar H-Series processor. This will automatically initiate a connection attempt.

Baud Rate

For RS-232 Control Mode only. Select the Baud Rate of the NovaStar H-Series processor. This will automatically initiate a connection attempt.

Data Bits

For RS-232 Control Mode only. Select the Data Bits of the NovaStar H-Series processor. This will automatically initiate a connection attempt.

Stop Bits

For RS-232 Control Mode only. Select the Stop Bits of the NovaStar H-Series processor. This will automatically initiate a connection attempt.

Parity

For RS-232 Control Mode only. Select the Parity of the NovaStar H-Series processor. This will automatically initiate a connection attempt.

Fan

Indicator LED is set high when any of the fans on the processor are in fault.

Voltage

Indicator LED is set high when the processor reports a voltage fault.

Temperature

Indicator LED is set high when the processor reports a temperature fault.

Genlock Signal

Indicator LED is set high when the processor detects genlock signal.

Device Name

Read only text that displays the processor's device name.

Device ID

Read-only text that displays the processor's device ID.

Connection Status

Displays the current connection status:

  • OK: Plugin is connected to the unit.
  • FAULT: Indicates a communication problem with the device, such as trying to connect to the wrong model.
  • MISSING: No device is found at the specified IP address or with the specified serial settings.

Input Status

Note: Available controls depend on the model selected in the Properties.

Slot

Read-only text that displays the input slot number.

Connector Type

Read-only text that displays the input slot's card interface configuration. This will display the interface configuration in the following form: "<interface 1> | <interface 2> | <interface 3> | ...". If no card is inserted, it should display "No Card Inserted".

Video Signal Status

Indicator LEDs that display the current input signal status. Each slot has up to 4 LEDs to represent the 4 possible interfaces on the card. If the LED is high, this means the corresponding input exists and is currently connected. If the LED is low, this means the corresponding input exists but is not currently connected. If the LED is disabled, this means the corresponding input does not exist (for example, LED 3 on an input card with 2 interfaces).

Output Status

Note: Available controls depend on the model selected in the Properties.

Slot

Read-only text that displays the output slot number.

Connector Type

Read-only text that displays the output slot's card interface configuration. If no card is inserted, it should display "No Card Inserted".

Video Signal Status

Indicator LEDs that display the current output link status. Each slot has up to 20 LEDs to represent the 20 possible interfaces on the card. If the LED is high, this means the corresponding output exists and currently has link. If the LED is low, this means the corresponding output exists but currently does not have link. If the LED is disabled, this means the corresponding output does not exist.

Screen Control

Screen Selection

Combo box used to select which screen to control. Screen selection affects all control for presets, layer routing, and screen controls.

Refresh Screen Info

Trigger button refreshes all screen info, including states of screen controls, presets, and layer routing.

Screen Blackout

State Trigger button fades the currently selected screen to black.

Screen Freeze

State Trigger button freezes the currently selected screen.

Screen Clear

Trigger button clears the currently selected screen.

Test Pattern Enable

State Trigger button enables a test pattern on the currently selected screen. If no test pattern is selected on the Test Pattern control, the Test Pattern control will default to "Black".

Test Pattern

Combo box used to select a test pattern. When a test pattern is selected, the Test Pattern Enable button should go high.

Screen Brightness

Combo box used to set output image brightness. This can be set in intervals of 20 from 0 to 100 and will report the currently set output image brightness.

Contrast

Fader to set the contrast of the currently selected screen. Allowed values are 0-100.

Brightness

Fader to set the brightness of the currently selected screen. Allowed values are 0-100.

Saturation

Fader to set the saturation of the currently selected screen. Allowed values are 0-100.

Hue

Fader to set the hue of the currently selected screen. Allowed values are -180-180.

Color Temperature

Fader to set the color temperature of the currently selected screen. Allowed values are 2000-10000.

Eye Save Mode

Toggle button to turn Eye Save Mode on or off. Turning this on disables Contrast, Brightness, Saturation, Hue, and Color Temperature controls.

Image Mode

Combo box used to set image mode. Choose between "Standard", "Document", "Conference", "Video".

Note: Any changes made to Contrast, Brightness, Saturation, Hue, and Color Temperature while on a particular Image Mode will "save" those settings for that Image Mode. For example, if Image Mode is set to "Standard", and Contrast is set to 25, the next time Image Mode is set to "Standard", Contrast will be 25.

Presets

Note: Besides the Current Screen Selection, the amount of these controls depends on the number of presets set in the properties.

Current Screen Selection

Read-only text that displays the currently selected screen.

Name

Read-only text that displays the preset name.

Load

Trigger button to load the corresponding preset.

Layer Routing

Note: Besides the Current Screen Selection, the available controls depend on the number of layers set in the Properties.

Current Screen Selection

Read-only text that displays the currently selected screen.

Name

Read-only text that displays the layer name.

Current Source

Read-only text that displays the currently selected source for the corresponding layer. It will be in the following format: "Slot <slot number> - In <input number> <interface>".

Source

Combo box to select which source to assign to the corresponding layer. The items in the combo box will vary depending on which input cards are installed. Selections will be in the following format: "Slot <slot number> - In <input number> <interface>".
