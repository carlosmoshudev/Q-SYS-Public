PJLink

Use this plugin to connect to, control, and display information for any PJLink-capable projector. Select projector inputs, monitor lamp hours, and monitor connection status.

The plugin is compatible with Q-SYS Designer version 7.2 and above, and was tested with PJLink version 2.0.

Note: Plugin compatibility with connected equipment is dependent on the projector manufacturer's implementation of the PJLink protocol and the features they have included.

Prerequisites

Before you can configure the plugin, ensure that your PJLink-compatible projector is connected to power and reachable on your network. QSC recommends that you configure a static IP address for the device, as DHCP addresses can change.

Configuration Overview

  1. Drag the plugin into your schematic.
  2. Press F5 to save your design to the Core and run it, or press F6 to emulate your design.
  3. In your schematic, double-click the plugin to open its control panel.
  4. In the Setup tab:
    • Enter the IP address of the projector.
    • Enter the port for PJLink communication (default is 4352).
    • Enter the password for the projector (if security is used).
  5. Projector communication will start immediately when valid credentials are entered.
  6. Use the Control tab to control the projector. See Controls.

Properties

Is Managed

Select 'Yes' to add the plugin to your Inventory list, which makes the plugin's Status available for monitoring in Core Manager and Enterprise Manager. When enabled, you can then specify a Name and Location for the plugin, as with any other Inventory item.

For more information, refer to the Plugins topic in the Q-SYS Help.

Upppercase Messages

Set the plugin to send message commands in uppercase characters. Setting to "no" will change the commands to lowercase. Recieved upper and lower case messages will be processed in either mode. Data fields in the API will not be affected.

Poll Rate

Set the rate at which the PJLink plugin will query the projector for status information, from 1 to 30 seconds.

Power Off Polling

Set the PJLink plugin to continue ("Yes") or stop ("No") polling status information when a power-off state is detected. Only power status will be queried when set to "No".

Poll Errors

Sets whether the PJLink plugin should poll the projector for error status.

Poll Lamp

Sets whether the PJLink plugin should poll the projector for lamp usage hours.

Poll Filters

Sets whether the PJLink plugin should poll the projector for filter usage hours. Filter hours is a class 2 request and will not be polled if the projector responds with a class 1 type API status.

Poll Resolution

Sets whether the PJLink plugin should poll the projector for current and recommended input resolution. Resolution status is a class 2 request and will not be polled if the projector responds with a class 1 type API status.

Input Count

Sets how many of input controls will be displayed of each type. Set between 2 and 36 input buttons to display. Inputs that are not supported on the device will be hidden when the plugin connects.

Show Debug

Select 'Yes' to show the Debug Output window. For details, see the Debug Output topic in the Q-SYS Help.

Debug Print

Set the data printed to the Show Debug window.

  • None: (default) None of the below will be posted to the log debug output.

  • Tx/Rx: Transmission and Received Data messages will be shown.

  • Tx: Transmission Data messages will be shown.

  • Rx: Received Data messages will be shown.

  • Function Calls: Each function within the plugin will post a message when it is called.

  • All: All of the above messages.

Controls

Control

Power

Indicates the connected projector's current power status. Click the power button to turn the connected projector ON or OFF.

A/V Mute

Toggle buttons for status and sending a mute/unmute command. Audio, Video or both (A/V) can be set. Freeze image is available for PJLink class 2 projectors.

Inputs

Select an input for the projector to display. Each input type – RGB, Video, Digital, Storage, Network, Internal – can have up to four input buttons. Only the detected types and number of inputs are shown. The name for each input will show above the button on PJLink class 2 devices.

Status

Video Info

Shows the current input's resolution and recommended resolution for that input.

Error Status

Indicators for error status states of OK/Warning/Fault for each of the projector type errors: Fan, Lamp, Temperature, Cover, Filter and Other.

Note: Any non-OK state will cause the plugin status to change to "Compromised".

Lamp Hours

Displays the detected hours for Lamp 1 and Lamp 2. The plugin polls for updated values every second.

Lamp Replacement

Displays the recommended filter and lamp model. This information is queried on first connection to the projector, and is available from PJLink class 2 devices only.

Setup

Connection

Specify the IP address, port and password of the projector. If security is not used, leave the password blank. The default port for the PJLink API is 4352.

Tip: If the Connect button is enabled and a valid projector IP address is specified, the plugin connects automatically when you start the design.

Connection Status

Displays the current plugin connection status:

  • OK: Plugin is connected to the unit
  • COMPROMISED: Plugin is connected to the unit; Device is reporting errors.
  • FAULT: Check the credentials to connect to the device.
  • MISSING: No device is found at the specified IP address.

Projector Information

Displays the detected projector Model, Manufacturer, and Projector Name. This section automatically updates if the projector's information changes. The plugin displays "<field> not supported" if the projector does not support a particular field name.
