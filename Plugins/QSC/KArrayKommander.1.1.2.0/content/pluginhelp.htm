K-Array Kommander

Use this plugin to control K-Array Kommander amplifiers.

The plugin has been tested with Q-SYS Designer version 9.4+. Earlier versions may work, but are not supported

Note: See the K-Array website for product information and documentation.

Configuration Overview

  1. Drag the plugin into your design.
  2. Select your model of amplifier in the Properties pane.
  3. Run the design and enter the IP address of your device on the setup tab
